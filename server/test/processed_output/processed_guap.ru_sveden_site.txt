Эти документы направлены на обеспечение условий доступности для инвалидов по зрению и других лиц с ограничениями жизнедеятельности.

___

Сайт ГУАП использует язык разметки HTML5 и адаптивный дизайн, что позволяет эффективно предоставлять информацию пользователям с ограниченными возможностями здоровья.
